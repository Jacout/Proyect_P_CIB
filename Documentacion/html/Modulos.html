Modulos package¶

Submodules¶

Modulos.Crearbase module¶

En este modulo se recibe los directorios con los archivos que se guardaron.

Para posterior mandar cada ruta de los directorio para obtener los valores hash de cada uno.

Modulos.Dominios_VT module¶

En este modulo se consulta en la api de Virus total que el sitio web ingresado sea seguro o no.

Modulos.Encriptado_files module¶

Modulo el cual se encripta los archivos generados en reportes por txt, y los exporta.

Modulos.metadata module¶

Modulo el cual obtiene los metadatos de los archivos descargados:

  • PDF

  • IMAGENES

Modulos.mod_beatifulsoup_img module¶

Modulo cual realiza el scraping del sitio web ingresado, obtiene imagenes principalmente.

Modulos.mod_beatifulsoup_pdf module¶

Modulo cual realiza el scraping del sitio web ingresado, obtiene archivos pdf principalmente.

Modulos.mod_beatifulsoup_url module¶

Modulo cual realiza el scraping del sitio web ingresado, obtiene url’s principalmente.

Modulos.pdfMaker module¶

Modulo el cual a partir de los reportes de salida en txt los convierte en pdf.

Modulos.portsv2 module¶

Modulo que con ayuda de socket el escaneo de puertos.
